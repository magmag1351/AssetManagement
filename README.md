# AssetPlanner - 資産管理 Web ダッシュボード

クレジットカード利用明細および銀行口座明細から、月ごとの収支管理およびジャンル別支出分析を行うモダンな Web GUI 資産管理アプリケーションです。マルチプロファイル管理に対応しています。

---

## 概要

本システムは `resources/` フォルダ内に配置された各種クレジットカード明細（Vpass CSV形式等）および銀行口座明細（NetBank CSV形式等）を自動読み込み、重複のない一元化収支データベース (`db/BalanceMaster.csv`) を構築します。

直感的な Web GUI ダッシュボードを通じて、収支のグラフ分析、取引明細の検索・フィルタリング、ジャンル分類ルールのカスタマイズ、および複数プロファイルの切り替え・管理が可能です。

---

## 主な機能

1. **マルチプロファイル管理システム (`👤 プロファイル切り替え`)**
   - 画面左下のプロファイルカードをクリックしてモーダルを起動。
   - 個人用、事業用、家族用など、複数のプロファイルをワンクリックで切り替え。
   - 新規プロファイルの追加および不要になったプロファイルの削除に対応。

2. **モダンな Web GUI ダッシュボード (`http://localhost:8000`)**
   - **KPI カード**: 収支差額 (Net Balance)、総収入、総支出、カード引落除外額をリアルタイム表示。
   - **月次収支推移チャート**: エリアグラデーション付き収入・支出ラインチャート (Chart.js)。
   - **分割された円グラフ**: 「ジャンル別」、「クレジットカード別」、「決済手段別（カード vs 口座直振・チャージ）」の独立したドーナツチャート。

3. **取引明細エクスプローラー & `Cmd + K` ショートカット**
   - 摘要やカード名でのリアルタイム検索および月・ジャンルによる絞り込み。
   - インラインジャンル編集（テーブル内のジャンルバッジをクリックして即時変更可能）。

4. **拡張性の高い明細パーサー構成**
   - Strategy パターンを採用したプラグイン構造。新しいカード会社や銀行の明細CSVが追加された場合でも、パーサークラスを追加するだけで拡張可能です。

5. **ジャンルのカスタマイズ画面 (`config/genre_rules.json`)**
   - GUI画面上で前方一致ルール (`prefix_rules`) や部分一致ルール (`keyword_rules`) を自由に追加・編集し、ワンクリックで既存明細を再分類可能です。

6. **二重計上の防止機能**
   - 銀行口座明細上のクレジットカード引き落としデータ（例: `ミツイスミトモカ－ド`）は「内部振替」として自動識別され、二重計上を防止します。

## 使い方

### 1. Web GUI ダッシュボードの起動および停止

#### 🚀 起動方法
以下のコマンドを実行して Web サーバーを起動します（追加の `pip` インストールは不要です）。

```bash
python3 main.py gui
```

起動後、ブラウザで **`http://localhost:8000`** を開きます。

#### 🛑 シャットダウン（停止）方法
- **通常停止**: サーバーを実行しているターミナル画面で **`Control + C`** (`Ctrl + C`) を押します。
  > ※ Macの「コピー (⌘ Command + C)」ではなく、**`Control + C`** キーで停止させます。
- **バックグラウンドにプロセスが残っている場合 / ポート解放**:
  ```bash
  lsof -ti :8000 | xargs kill -9
  ```

---

### 2. プロファイルの切り替え・追加・削除
画面左下のプロファイルカード（例: 「上杉 健太」）をクリックしてプロファイル管理モーダルを開き、切り替えや新規プロファイルの作成・削除を行います。

---

### 3. CLI コマンド（オプション）
```bash
# CLIでのインポート
python3 main.py import

# CLIでのレポート表示
python3 main.py report

# ユニットテストの実行
python3 -m unittest discover tests
```
